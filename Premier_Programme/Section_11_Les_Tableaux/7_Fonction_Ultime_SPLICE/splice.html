La Fonction Ultime - SPLICE

La fonction SPLICE permet d'ajouter, de remplacer ou de supprmer des éléments dans un tableau.

Comme pour la concaténation, cette fonction ne s'utilise que dans les tableaux simples et à plusieurs dimensions.

La fonction splice demande plusieurs paramètres :

  • le premier est l'indice à partir duquel commencer (Ex: (1))
  • le second est le nombre d'éléments à supprimer à partir de cet indice:
    (Ex: (1, 2) => Ici on supprime 2 éléments à partir de l'élément ayant l'indice 1)
  • le troisième est le ou les Items que l'on veut rajouter (ils sont illimités)
    (Ex: (1, 2, "random", "pie") => Ici on supprime 2 éléments à partir de l'élément ayant l'indice 1)

    • le 3ème paramètre n'est pas obligatoire si on ne rajoute rien dans le tableau.

Pour les tableaux à plusieurs dimensions, on peut soit supprimer une ligne complète du tableau,
soit un élément d'une ligne du tableau.
