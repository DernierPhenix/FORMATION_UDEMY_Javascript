Quand on ne connaît pas le nombre d'éléments d'un tableau, on peut utiliser la fonction

LENGTH. Cette fonction va alors afficher le nombre d'éléments contenus dans le tableau.

Ensuite pour avoir le dernier élément par exemple on peut écrire ceci :

console.log(monTableau[monTableau.length - 1]); Voir dans le code du tableau Simple.
