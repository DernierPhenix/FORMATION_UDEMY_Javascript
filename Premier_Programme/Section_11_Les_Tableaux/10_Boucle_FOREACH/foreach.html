La Boucle FOREACH

La boucle FOREACH permet de parcourir un tableau et d'afficher les éléments de ce tableau.

Elle fait appel à la fonction anonyme. Voir le code (1ère Méthode)

On peut simplifier la boucle en l'écrivant sur une seule ligne et en utilisant les 'FONCTIONS FLÉCHÉES'. Voir le code (Autre Méthode)
