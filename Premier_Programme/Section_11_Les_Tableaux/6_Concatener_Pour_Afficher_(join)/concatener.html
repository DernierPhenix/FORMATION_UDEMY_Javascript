Concaténer Pour Afficher - La Fonction JOIN

En utilisant la fonction JOIN, Js transforme notre tableau en une seule et même chaîne de caractère.

Voir le code pour exemple ET dans la console du navigateur F12 pour le résultat

Cette fonction n'est utilisable qu'avec les tableaux simples et à plusieurs dimension.

Dans un tableau à plusieurs dimension, toutes les "lignes" ayant un index différents sont .

également transformées en une seule chaîne de caractères.

Si on souhaite change de séparateur, cette séparation apparaîtra entre chaque nouvel index (chaque lgne).

Pour changer le séparateur sur une ligne du tableau, il faut simplement mettre son index entre crochets

avant le '.join'
