La Boucle FOR...IN

La boucle FOR...IN permet d'afficher les INDEX de tous les éléments d'un tableau dans la boucle.

On va donc définir une constante qui va enregistrer les INDEX des éléments

du tableau, puis on peut demander, dans la console, d'afficher l'élément correspondant à

l'index

Voir le code pour le résultat ou dans la console du navigateur (F12)
