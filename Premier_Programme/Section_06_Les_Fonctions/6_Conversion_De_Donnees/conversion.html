Conversion de Données

Le But de l'exercice au départ est de créer un programme qui demande l'âge de l'utilisateur et

ensuite affiche vous avez "age" ans. Pour cela, on écrit le code suivant :

function demanderAge() {
let age = prompt("Entrer votre âge : ");
alert("Vous avez " + age + " ans !");
}
demanderAge();

Maintenant on crée une fonction qui va prévoir l'âge dans 1, 2 , 10 ans.

On pourrait écrire le même code mais modifier la partie alert comme suit,

alert ("Vous avez " + (age + 1) + " ans")

Cela ne fonctionnera pas, on aura un message qui affichera qu'on a 301 ans si on rentre 30.

Cela est dû au fait que Js prend la valeur que l'on rentre dans le prompt comme une chaîne de caractères.

Pour obtenir notre résultat, on va utiliser des fonctions déjà existantes dans Js comme :

- Number() : Converti une chaîne de caractères en nombre (avec ou sans virgule). CELUI QUI EST RECOMMANDÉ.

- parseInt() : Converti une chaîne de caractères en entier. ATTENTION 35,5 sera converti en 35.

- parseFloat() : Converti une chaîne de caractères en nombre à virgules.

ATTENTION : Dans les parenthèses , on écrit le nom de la variable.

Pour transformer un nombre en chaîne de caractères, on utilisa la fonction toString.
