Les Conditions Ternaires

Une condition ternaire permet de mettre une condition mais une seule ligne.

Elle s'écrit de la façon suivante:

[condition] ? [True] : [False]

Exemple :

let x = 5;
x > 3 ? console.log("X est supérieur à 3.") : console.log("X est inférieur à 3.");

Autre Exemple :

let gareDeDepart = "Valenciennes";
let gareDarrivee = prompt("Quelle est votre destination ?") || "Gare du Nord";
let chauffeur = "Jean-Pierre";
if ((gareDeDepart != "" || gareDarrivee != "") && chauffeur != "") {
alert("Le train va démarrer à destination de " + gareDarrivee + ".");
} else {
alert("Le train ne peut pas démarrer sans chauffeur")
}

Dans cet exemple on a définit la variable gareDarrivee dans un Prompt.

Si l'utilisateur, indique une destination alors le message avec sa destination apparaîtra dans une boîte de dialogue.

Si l'utilisateur, n'indique pas de destination alors le message apparaîtra dans une boîte de dialogue avec la valeur par défaut

qui se trouve après le double pipe (Gare du Nord).
