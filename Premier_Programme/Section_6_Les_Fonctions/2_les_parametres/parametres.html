Les Paramètres de la Fonction

Les paramètres d'une fonction sont considérées comme des variables modifiables.
On peut mettre plusieurs paramètres à une seule fonction.
Ex : function name(params){
}
=> function addition(nombreUn, nombreDeux, nombreTrois){
