Variables Locales

Une variable locale est une variable définie dans une fonction

Elle n'est utilisée que lorsqu'on fait appel à la fonction

Variables Globales

Une variable globale est une variable définie en dehors d'une fonction

Lorsque la variable globale n'est pas definie, on peut faire appel à la même variable locale (voir Exemple 1)

Lorsque la variable globale est définie, on peut l'afficher en l'appelant après la fonction (voir Exemple 2)
